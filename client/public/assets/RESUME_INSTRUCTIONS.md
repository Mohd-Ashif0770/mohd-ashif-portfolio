# Resume File Instructions

## Location
The resume file should be placed at: `client/public/assets/Resume.pdf`

## File Name
- **Required**: `Resume.pdf`
- The download button links to `/assets/Resume.pdf`

## How to Add Resume

1. Copy your resume PDF file to this directory
2. Rename it to `Resume.pdf`
3. The download button in the About section will automatically work

## Note
- File must be named exactly `Resume.pdf` (case-sensitive)
- Supported format: PDF
- The download attribute in the link will trigger the download

## For Development
In development, the file will be served from `/assets/Resume.pdf`

## For Production (Vercel)
The file will be served from `/assets/Resume.pdf` after deployment
